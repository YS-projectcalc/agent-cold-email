# ColdStart (working name)

Multi-tenant cold-email infrastructure platform built to be **operated by the customer's own coding agent** (Claude Code / Codex). One signup + one token; the agent does everything else: provisions branded lookalike domains, warms mailboxes, runs sequences, manages replies. We orchestrate wholesale email infrastructure underneath; the customer's agent is the brain.
